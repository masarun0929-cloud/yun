更新手順

  1. ローカル管理画面またはD1 Consoleで歌枠データを更新する。
  2. 管理画面の「静的JSONを生成」で docs/data/*.json を更新する。
  3. 変更をGitへcommit/pushしてCloudflare Pagesへ反映する。
  4. 反映後、このページで件数と最新日付を確認する。
